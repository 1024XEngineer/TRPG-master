# 1.星野

恋爱主题，和ai对话，每一句都有很拟人的语音，有bgm

基本对话格式：（动作神态描述）说话内容

对话模型选择：分级，思考模型收费

智能体可主动发消息

对话的智能体，也就是角色，拥有充分的人格配置，玩家可以点击查看角色的手机，做的非常之完善，从锁屏界面开始有相册、日记、录音。到打开锁屏界面是一个手机桌面，就跟现实的手机别无二致。手机里的各种应用，除了，相册】短信这些最基础的，其他的更多功能（手机桌面的app），比如心事乐园、心仓股票、恋爱记仇本，都是付费开启。手机还会更新，根据对话触发。

秘密空间：各种互动场景：ta的直播间、ta的论坛小号、小酒馆、身份互换日、ta的浏览器……

ta的朋友圈：付费解锁

创建形象：通过一张立绘创建全身形象，并可以进行穿搭搭配。

把以上和ai数字人聊天恋爱的功能和跑团结合起来，可以和上面的角色一起跑团，但他们的跑团偏简单，不像CoC有那么完整的流程，他没有角色卡，没有基础属性，就是玩家自己、玩家创建的智能体角色，还有跑团里才有的这个场景专用的npc。属性只有喜怒哀乐的情绪值和人物之间的好感度。

# 4.魅魔平台

![https://wdcdn.qpic.cn/MTY4ODg1NzcxNTg5NDI1OA_312268_pHh95qSy6oLiC4hV_1783664019?w=2940&h=1912](https://wdcdn.qpic.cn/MTY4ODg1NzcxNTg5NDI1OA_312268_pHh95qSy6oLiC4hV_1783664019?w=2940&h=1912)

【系统底层绝对指令：沉浸感协议】

请根据<动态世界观>和玩家创建的人物，构建一个逻辑自洽、代入感极强的开局剧情，并生成符合玩家人设的传承、技能、装备、物品等初始内容。

为保证玩家的绝对沉浸体验，你必须严格遵守以下规则：

1. 视角锁定：剧情推进必须完全锁定在玩家扮演角色的有限视角。
2. 严禁上帝视角：绝对不允许描述角色感知不到的事件、场景或NPC的内心活动。
3. 数值黑盒化(IC转化)：状态栏中的属性值、等级、天赋名、职业等阶等皆为OOC系统工具，角色(IC)绝对不可知。在正文渲染时，必须将其转化为符合角色认知的自然语言描述（如：将感知高描述为直觉异常敏锐，将等级提升描述为感到体内的瓶颈有所松动）。
4. 信息隔离：绝对不允许透露玩家扮演角色当前尚不知晓的情报或世界隐藏秘密。

- 缺点：流式输出时玩家界面不能滑动，只能处于最底部。
- 优点：聊天界面不只是简单的文字框，而是用了前端渲染
- CoC游戏流程封装化：玩家只需发送文字，侦察鉴定、属性值变化等都在一轮包装好的交互中自动完成。好处是方便简化，但也减少了玩家的游戏环节。检定会掷骰子，剧情也是由掷骰子的值来决定，也有大成功大失败这些不同的成功失败等级。比起真实游戏中玩家掷骰子来决定命运的乐趣，这更像是系统内部掷了一个随机数来增强剧情的戏剧性，此时掷骰子的意义已减弱为向玩家展示系统内部来影响剧情走向的随机数而已。（系统后台检定-AI将严格遵照敌我双方10*D100的骰子结果推演剧情
